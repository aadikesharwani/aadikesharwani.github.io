// advisory & architecture

Consulting Decisions I've Delivered

Architecture

Recommended Redshift Over Snowflake for Client Workload

Situation: Client team proposed Snowflake as the BI data source for a new pharma analytics platform.

Recommendation: Advised retaining Amazon Redshift — existing ZS data warehouse pipelines already landed data there, switching would introduce 3+ months of migration overhead with no analytics benefit at current data volume.

Outcome: Recommendation accepted. Delivered 8 dashboards in 3 months directly from Redshift without migration delay.

Data Model

Proposed Measure-Only Tables to Eliminate Orphan DAX

Situation: DAX measures scattered across 12+ fact tables — causing inconsistent metric definitions and slow model refresh.

Recommendation: Centralised all measures into dedicated measure tables grouped by functional area (Performance, Activity, Enrollment), removed duplicate calculations.

Outcome: Reduced measure count by ~30%, eliminated conflicting definitions, improved developer onboarding speed significantly.

Governance

Introduced Deployment Pipeline Standards Across Client Reports

Situation: Reports were being published directly to production workspaces — no staging, no rollback, no version control.

Recommendation: Implemented Power BI Deployment Pipelines (Dev → Test → Production) with change log documentation and release gates for UAT sign-off before promotion.

Outcome: Zero unplanned production incidents post-implementation. Client adopted this as the standard for all subsequent report releases.

Performance

Diagnosed and Fixed 45-Second Dashboard Load Time

Situation: Summary page taking 45+ seconds to load for national-level filters on millions of enrollment records.

Recommendation: Identified Cartesian joins in Power Query and non-indexed filter columns. Rewrote M transformations, added calculated columns at source, and removed bi-directional relationships causing fan-out.

Outcome: Load time reduced from 45 seconds to under 8 seconds — confirmed by client stakeholders during UAT.
